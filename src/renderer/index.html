URL Routing

DIRECT .mkv .mp4 .webm .avi .ts etc. → skips yt-dlp, MPC-HC streams the file directly over HTTP
YT-DLP YouTube, Crunchyroll, Twitch, Bilibili → yt-dlp resolves hidden stream URL → MPC-HC plays it
FALLBACK If yt-dlp fails or times out → URL passed directly to MPC-HC anyway
