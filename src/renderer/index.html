How it works

This app embeds Chromium (via Electron) as its video engine — the same engine as Google Chrome. It handles:

  • Direct files: .mp4 .mkv .webm .mov .avi .ts .m4v
  • HLS streams: .m3u8 adaptive playlists (hls.js)
  • Any site supported by yt-dlp (URL is resolved to a raw stream)
  • Picture-in-Picture · playback speed · volume · keyboard shortcuts
